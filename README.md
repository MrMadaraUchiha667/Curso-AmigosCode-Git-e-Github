## 1. Introdução ao Git

### O que é Git?

Git é um sistema de **controle de versão distribuído** utilizado para acompanhar alterações em arquivos, principalmente código-fonte.

Com Git é possível:

- Controlar versões de projetos.
- Trabalhar em equipe.
- Criar históricos de alterações.
- Recuperar versões anteriores.
- Criar branches para desenvolvimento paralelo.
